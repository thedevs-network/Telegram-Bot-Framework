# Guard bot V2

## Usage:

	node . start <token>

Start the bot with `<token>`


	node . enable <name>

Enable plugin `<name>`


	node . disable <name>

Disable plugin `<name>`


	node . set <plugin> <option> <value>

Set `<option>` in `<plugin>` to `<value>`


	node . unset <plugin> <option> <value>

Unset `<option>` in `<plugin>` to `<value>`
